******** Resquisitos Funcionais ********


* O sistema permitirá acesso de administrador
* O sistema permitirá acesso de gerente
* O sistema permitirá cadastro de clientes
* O sistema permitirá cadastro de funcionários
* O sistema permitirá gerenciamento de pagamento dos clientes
* O sistema autorizará a entrada de clientes na academia
* O sistema gerará um relatório mensal contendo clientes adimplentes e inadimplentes 
* O sistema permitirá elaboração de balanço mensal
* O sistema gerará relatório de balanço mensal
* O sistema gerará uma senha exclusiva por cliente
* O sistema permitirá cadastro de treinos físicos
* O sitema permitirá cadastro de avaliações físicas dos clientes
* O sistema associará treinos físicos a avaliações físicas para cada cliente
* O sistema gerará relatório contendo resultado da avaliação física do cliente
* O sistema marcará datas para as avaliações físicas do cliente
* O sistema marcará datas para períodos de treino
* O sistema manterá dados de funcionários
* O sistema permitirá cadastro de equipamentos da academia
* O sistema marcará datas de manutenção para cada equipamento
* O sistema permitirá cadastro de aulas de ginástica
* O sistema permitirá agendamento de aulas de ginástica
